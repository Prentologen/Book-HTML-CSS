Стиль списку.

У дев'ятому розділі підручника HTML ми з Вами вже знайомилися з таким важливим елементом як список і справді хороший засіб для структуризації даних. Однак списки організовані одними засобами HTML дуже убогі в плані дизайну і не радують око людини.

В этой главе мы немного поколдуем над списками с помощью свойств CSS. Так мы будем говорить о стилях списка

Вид маркера в списке.

Если вы помните, то в чистом HTML вид маркера в списке определял атрибут type и одно из его возможных значений, в CSS данную задачу берёт на себя свойство: list-style-type которое, в свою очередь, тоже имеет свои стандартные значения определяющие вид маркера как всего списка сразу, так и его отдельного "пункта".

Значення list-style-type:

  • disc - Диск. (За замовчуванням для <ul>)
  • circle - Порожній коло.
  • square - Квадрат.
  • decimal - Арабські цифри. (За замовчуванням для <ol>)
  • lower-roman - Малі римські цифри.
  • lower-alpha - Малі літери.
  • upper-roman - Заголовні римські цифри.
  • upper-alpha - Великі букви.
  • none - Маркер відсутній.

Приклад:

<!DOCTYPE HTML>
<html>
<head>
<title>Вид маркера в списку </title>
</head>
<body>
<ul style="list-style-type: square">
<li>Пункт 1.
<li>Пункт 2.
<li style="list-style-type: circle">Пункт 3 (особливий).
</ul>
<ul style="list-style-type: upper-roman">
<li>Пункт 1.
<li>Пункт 2.
<li>Пункт 3.
</ul>
</body>
</html>

Призначений користувачем маркер малюнок.

Найбільш цікавим CSS інструментом для роботи зі стилем списку є, на мій погляд, можливість замість стандартних "нудних" маркерів описаних вище використовувати свої власні нестандартні зображення - невеликі малюнки, які вписуються в загальний дизайн Вашого сайту.

Це завдання виконує властивість list-style-image яке визначає в якості маркера списку якесь графічне зображення із зазначенням шляху до нього.

Значень даного властивості всього два:

  • none - Скасовує графічне зображення маркера.
  • url - Шлях до файлу з малюнком маркера.

Шлях до малюнка після url вказується в круглих дужках.

Ось так:

list-style-image: url(graphics/marker.gif)

- Такий запис буде говорити про те, що поруч з документом є папка graphics в якій лежить файлик-малюнок "  " - під назвою marker.gif

Тепер попрубуем зробити так, що б кожен пункт нашого списку був промаркований цим малюнком. Дивимося приклад:

<!DOCTYPE HTML>
<html>
<head>
<title>Нестандартний маркер-малюнок </title>
</head>
<body>
<ul style="list-style-image: url(graphics/marker.gif)">
<li>Перший улюблений пункт.
<li>Другий улюблений пункт.
<li>І не менш улюблений третій пункт.
</ul>
</body>
</html>

Стиль обтікання маркера списком.

Властивість list-style-position вказує браузеру на те, як слід відображати текст в списку щодо його маркерів. За замовчуванням маркери знаходяться в стороні від тексту списку, але можна зробити так, що вони будуть обтікати текстом.

Можливих значень властивості list-style-position всього два:

  • outside - Маркер знаходиться в стороні від списку. (за замовчуванням)
  • inside - Маркер обтікается текстом.

Приклад для наочності:

<!DOCTYPE HTML>
<html>
<head>
<title>Обтікання маркера текстом </title>
<style type="text/css">
body {
margin: 0px;
background: #e8e8e8
}
div {
width: 300px;
height: 200px;
float:left;
margin: 10px;
padding: 10px;
border: RGB(25, 125, 25) 2px ridge;
background: #fff
}
h3 {
text-align: center
}
</style>
</head>
<body>
<div>
<h3>Тут маркер обтікає текстом:</h3>
<ul style="list-style-position:inside">
<li>Пункт, в якому йдеться про те, що добре б було зробити, щось там, де це щось ще не зроблено.
<li>Пункт, в якому йдеться про те, що непогано б було зробити, щось там, де це щось ще не зроблено.
</ul>
</div>
<div>
<h3>А тут немає:</h3>
<ul style="list-style-position:outside">
<li>Пункт, в якому йдеться про те, що добре б було зробити, щось там, де це щось ще не зроблено.
<li>Пункт, в якому йдеться про те, що непогано б було зробити, щось там, де це щось ще не зроблено.
</ul>
</div>
</body>
</html>

list-style

Базова властивість list-style використовується, коли стилю списку необхідно одночасно привласнити кілька значень. Може мати від одного до трьох значень з властивостей застосовуваних до стилю списку, в будь-якій послідовності через пропуск.

Всі три властивості та їх можливі значення ми розглянули вище, тому повторюватися не буду, а просто перерахую їх:

  • list-style-type - Вид маркера в списку
  • list-style-image - Нестандартний маркер малюнок
  • list-style-position - Стиль обтікання маркера списком

Якщо в голові залишилися якісь прогалини можете повернутися і перечитати.

Приклад:

<!DOCTYPE HTML>
<html>
<head>
<title>Стиль списку </title>
</head>
<body>
<div style="width: 250px;">
<ul style="list-style: url(graphics/marker.gif) inside">
<li>- Цей список використовує в якості маркера малюнок.
<li>- Текст цього списку обтікає маркер.
</ul>
</div>
</body>
</html>

Корисні поради:

  • Списки поряд з заголовками, виділеним текстом і картинками, є елементами, які приковують погляд відвідувача при побіжному перегляді сторінки. Вміле і ненав'язливе розміщення таких елементів є запорукою того, що відвідувач зануриться в читання Вашого документа більш глибоко.

  • Так як списки є чудовим інструментом для структуризації даних, а властивості CSS дозволяє створити практично будь-який їх дизайн, використовуйте списки, крім їх основного призначення, як "меню" - навігації по сайту, де кожен пункт є посиланням на ту чи іншу сторінку Вашого сайту .
